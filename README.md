# ตัวอย่างการใช้ Express + NodeJs + MongoDB -> RabbitMQ

## install
```npm install```
* ทั้ง 2 ตัว

## Development
```cd receiver```
```npm run dev```
* เปิดพร้อมกัน 2 ตัว
```cd sender```
```npm run dev```
* test POST PUT DELET in POSTMAN
* Use RabbitMQ server https://www.cloudamqp.com/
